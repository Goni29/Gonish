# Gonish Analytics Notes

## 구현 기준

- 이 분석 로직은 로그인 계정이 없는 환경을 전제로 합니다.
- `visitor_id`는 동일 브라우저 환경의 재방문을 추정하기 위한 익명 랜덤 ID입니다.
- `session_id`는 한 번의 방문 세션을 나타내는 익명 랜덤 ID입니다.
- IP 주소를 방문자 식별자로 사용하지 않으며, 원본 IP를 `analytics_events`에 저장하지 않습니다.
- URL과 referrer는 query string을 제거해 저장합니다.
- 관리자 세션은 `/api/analytics`에서 실제 세션 검증 후 집계에서 제외합니다.
- 원본 이벤트 보관 기간은 90일, 집계 데이터 보관 기간은 12개월을 권장합니다.

## 개인정보처리방침 문구 초안

서비스 이용 과정에서 방문 일시, 페이지 이용 기록, 페이지별 체류 시간, 유입 경로, 브라우저 정보, 익명 방문자 ID 및 세션 ID가 자동 생성되어 수집될 수 있습니다.

익명 방문자 ID는 서비스 이용 통계 분석과 동일 브라우저 환경에서의 재방문 여부 추정을 위해 사용하는 무작위 식별값이며, 로그인 정보, 이름, 이메일 등 개인을 직접 식별할 수 있는 정보와 결합하지 않습니다.

브라우저 저장소 또는 쿠키를 삭제하거나, 시크릿 모드를 사용하거나, 다른 브라우저 또는 다른 기기를 사용하는 경우 새로운 방문자로 인식될 수 있습니다.

원본 IP 주소는 분석 목적의 방문자 식별자로 사용하지 않으며, 분석 이벤트 저장소에 원본 IP를 저장하지 않습니다.
